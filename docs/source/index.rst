Red Stork
=========

Yet another PDF parser. This one is based on _PDFium engine.

.. _PDFium https://pdfium.googlesource.com/pdfium/


Quick Start
^^^^^^^^^^^

Sample::

    from redstork import PDF_Document

    doc = PDF_Document('sample.pdf')
    print('Number of pages:', len(doc))
